Singapore GeBIZ Scrapers

The Singapore folder contains Playwright and Selenium GeBIZ scrapers plus MDT export helpers.

Google Drive source copies were extracted under Singapore/drive_extracts/Singapore_GeBIZ for provenance and comparison with the active scripts.

Scripts

  • Singapore/scripts/gebiz_scraper.py: Playwright scraper.
  • Singapore/scripts/gebiz_scraper_selenium.py: Selenium fallback.
  • Singapore/scripts/latam_spec_defaults.py, mdt_export.py, mdt_schema.py: output helper modules.
  • Singapore/drive_extracts/Singapore_GeBIZ/: Drive extracted scaffold README, requirements, GeBIZ scraper, and notebook.

Run

python Singapore/scripts/gebiz_scraper.py --date-from 2026-01-01 --date-to 2026-03-31 --query "medical" --max-pages 10 --output-target Singapore/output
