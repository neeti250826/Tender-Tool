Kuwait CAPT Scraper

The Kuwait folder contains a CAPT winning-bids scraper plus PDF helper utilities.

Scripts

  • Kuwait/CAPT/capt_awarded_scraper.py: scrape CAPT winning bids and retain medical rows.
  • Kuwait/CAPT/capt_pdf_helper.py: inspect local CAPT PDFs with OCR/rendering support.
  • Kuwait/CAPT/capt_verified_seeds.py: verified rows and medical-scope exclusions.

Run

python Kuwait/CAPT/capt_awarded_scraper.py --date-from 2024-01-01 --date-to 2026-06-25 --max-pages 250 --headless --translate
