Malaysia MOH Tender Documentation

Google Drive extraction added the Malaysia MOH scraper, notebook wrapper, output helpers, and SOP document.

Status

Malaysia/scripts/moh_scraper.py is the main runnable scraper. It targets public MOH tender results at https://www.moh.gov.my/en/tender/tender-results, downloads or parses tender result pages, and writes standard spec/MDT outputs using the included helper modules.

Run

python Malaysia/scripts/moh_scraper.py --date-from 2026-03-01 --date-to 2026-03-14 --query "digital" --filter-match any --start 0 --output-target Malaysia/output --project-name MDT_2026 --website-id MY_MOH --source-label "Malaysia MOH" --region EMEA

Drive Documents

  • Malaysia/docs/MOH_Tender_SOP.docx: MOH process document covering tender listings, PDFs, extraction fields, cleaning, classification, and outputs.
  • Malaysia/scripts/moh_scraper_notebook.ipynb: Drive extracted notebook wrapper.
