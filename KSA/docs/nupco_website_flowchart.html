NUPCO Website Flow Chart

This flow chart covers only the NUPCO website path for finding awarded tenders, opening tender details, checking attachments, and building the final awarded tender record.

NUPCO Website Only

1. Open NUPCO Tenders Page

Go to https://www.nupco.com/en/tenders/.

2. Choose Award-Related Status

Use only the award-relevant filters: Final Result and First Result.

3. Review Tender Listing

Scroll through the listing and open the tender card or Tender ID link for each relevant tender.

4. Open Tender Detail Page

Capture the tender metadata such as Tender ID, title, status, publication date, and submission deadline.

5. Check For Result Attachments

Look for Final Result, Preliminary Result, Announcement, or similar result files.

If Result Attachments Exist

  • Open the result PDF.
  • Collect supplier name and award outcome.
  • Check whether an Item List attachment is also available.
  • If yes, open it and collect item description, quantity, and UOM.

If Result Attachments Do Not Exist

  • Use the tender detail page only.
  • Record whatever tender information is visible on the website.
  • Mark item-level or supplier-level fields as unavailable if they are not shown.

6. Combine All Available Data

Merge tender detail data, result file data, and item list data into one awarded tender record.

Final Output: Awarded Tender Record
