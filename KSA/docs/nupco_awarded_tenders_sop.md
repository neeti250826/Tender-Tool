# NUPCO Awarded Tenders SOP

## Purpose

This SOP explains how a non-technical user can open the NUPCO tenders website, apply the correct filters, and locate awarded tender data.

## Website

- Main tenders page: `https://www.nupco.com/en/tenders/`

## What To Look For

On the tenders page, NUPCO shows status filters near the top of the listing.

The live filters currently visible are:

- `Show All`
- `direct purchase`
- `First Result`
- `Final Result`
- `Under Sudying`
- `Available`
- `Open/Updated`
- `Canceled`

For awarded tender data, the important filters are:

- `First Result`
- `Final Result`

These are the statuses that indicate award-related outcomes are available.

## Step-By-Step Process

### 1. Open the NUPCO tenders page

- Open your browser.
- Go to `https://www.nupco.com/en/tenders/`.

### 2. Go to awarded tenders only

- At the top of the tenders list, click `First Result`.
- Review the tenders shown under this filter.
- Then click `Final Result`.
- Review the tenders shown under this filter as well.

Note:
- `First Result` usually means preliminary or first-stage award information.
- `Final Result` usually means final awarded tender information.

If you want the most complete awarded data, check both filters.

### 3. Open an individual tender

- In the filtered list, click a tender card or the `Tender ID` link.
- This opens the tender detail page.

## Where The Awarded Data Usually Appears

On the tender detail page, awarded data is usually not fully shown as structured page text.

Instead, it is commonly available through attached files such as:

- `Final Result`
- `Preliminary Result`
- `Announcement`
- `Item List`
- related PDF attachments

## What Data You Can Usually Collect

From the tender page and its result files, you can usually get:

- Tender ID
- Tender title
- Tender status
- Publication or bid opening date
- Submission deadline
- Supplier name
- Award-related amount or value
- Awarded item details, if an item list file is attached

## Recommended Manual Review Order

For each awarded tender:

1. Open the tender from `Final Result` first.
2. Check whether there is a `Final Result` PDF attachment.
3. If not, check for `Preliminary Result` or similar result file.
4. Check whether there is an `Item List` attachment.
5. Use the result document for supplier and award outcome.
6. Use the item list document for item description, quantity, and UOM if available.

## Which Filters Not To Use For Award Data

These filters are usually not for awarded tenders:

- `Available`
- `Open/Updated`
- `Under Sudying`

These are generally active or non-final competition stages.

`Canceled` is also not awarded data.

## Practical Rule

If the goal is to find awarded tenders, use this order:

1. `Final Result`
2. `First Result`

Ignore the rest unless you need non-awarded tender activity.

## Output Expectation

When checking the NUPCO website manually, the awarded tender data is typically spread across:

- the tender listing status
- the tender detail page
- attached PDF result files
- attached PDF item list files

So the user should expect to open the tender and then open the related PDF attachments to see the full awarded information.

## Notes

- The website is scroll-based, so more tenders may load as you move down the page.
- Some awarded tenders appear only after scrolling further down the listing.
- `Final Result` is the best filter for final awarded outcomes.
- `First Result` is still useful and should also be checked because some tenders publish award-related information there.
