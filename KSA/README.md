# KSA Tender Scripts

KSA contains Clearstate-extracted Saudi award scrapers for NUPCO and Etimad.

## Scripts

- `scripts/nupco_awards_scraper.py` - NUPCO awarded tender scraper.
- `scripts/etimad_awards_scraper.py` - Etimad awarded tender scraper.
- `scripts/award_io.py` - shared award output helper.
- `scripts/validate_award_jsonl.py` - JSONL validation helper.

## Run Commands

```powershell
python KSA/scripts/nupco_awards_scraper.py
python KSA/scripts/etimad_awards_scraper.py
```
