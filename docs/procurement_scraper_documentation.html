Procurement Scraper Documentation

This report documents the current implementation and verified results for the country-level Python scrapers built from the assessed feasible public procurement sources, focused on medical, healthcare, equipment, consumables, and related tenders.

Date window: 2024-01-01 to 2026-06-01 Report date: 2026-06-01 Workspace: C:\Users\Neeti\OneDrive\Documents\TURMECA

Objective

Build Python scrapers in country/source folders for public procurement websites that are feasible to scrape for medical or healthcare-related tenders, keep outputs within the publication-date window from 2024-01-01 through 2026-06-01, populate awarded columns where the source exposes awarded data, duplicate tender-level data across multiple item rows, smoke-test one page first, and then run the scraper as broadly as the public source allows.
