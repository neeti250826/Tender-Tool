# Egypt - upa.gov.eg

- Portal: `https://upa.gov.eg/tenders/`
- Access: Public pages load normally without VPN or login. Supplier registration is required for procurement participation and some document access.
- Public granularity: Tender-level only on public pages. Public tender pages expose title, create date, start date, end date, booklet price, bid bond, and short descriptions.
- 2024+ awarded/complete/published count: `4` published tender pages verified from `2025-05-11` to `2026-01-23`. Awarded/completed counts were not publicly exposed.
- Tender-item extraction feasibility: `Low` from anonymous public pages; `Medium` only if registered supplier documents can be accessed.
- Notes: Public history page currently does not provide a usable award/completion result set.
